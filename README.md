Trabalho desenvolvido na disciplina de Programação Web pelo Profº Vinicius Bortolini realizado pelos alunos: Kauã Trudes, Elton Carlotto e Roger Bieberbach. 
O objetivo da atividade era implementar uma ideia de site com funcionalidade semelhante ao "Last.fm", onde era destacado artistas diariamente, seja por estar em alta, novos lançamentos, como memorial, etc.
